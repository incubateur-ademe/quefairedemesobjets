Comment réduire vos déchets ?

Retrouvez tous les conseils de l’ADEME sur le site Agir pour la transition. Si vous souhaitez aller plus loin dans votre démarche, vous pouvez calculer votre empreinte écologique avec Nos Gestes Climat .

D'où viennent nos données ?

Cet outil utilise les données agrégées par l'ADEME, il s'agit de données à l'échelle nationale.

Comment intégrer cet outil à votre site ou application ?

Cliquez sur le bouton " Intégrer" et copiez/collez le code proposé dans votre site Internet. Vous souhaitez réutiliser le code de cet outil ? Il est développé de manière ouverte (open source), le code est  disponible librement .

Comment contribuer au développement de cet outil ?

Vous pouvez répondre  à notre enquête . Votre participation est importante pour s'assurer que nous développons un outil pertinent et répondant à vos besoins.
