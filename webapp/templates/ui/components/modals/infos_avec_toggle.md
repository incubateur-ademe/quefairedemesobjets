# Modal Infos avec formulaire de contrôle

Permet de tester différentes configurations de la modal d'infos :

- `button_only`: affiche uniquement le bouton (sans la modal)
- `modal_only`: affiche uniquement la modal (sans le bouton)
- `button_text`: personnalise le texte du bouton

Par défaut (tous les flags à False), affiche à la fois le bouton et la modal.
