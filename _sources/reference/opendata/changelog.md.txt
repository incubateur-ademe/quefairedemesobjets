# Changelog du jeu de données Open Data : « Acteurs de l'économie circulaire - Longue Vie Aux Objets »

Ce document fait référence au jeu de données open data suivant :
[Acteurs de l'économie circulaire - ADEME](https://data.ademe.fr/datasets/longue-vie-aux-objets-acteurs-de-leconomie-circulaire)
Il ne trace pas l’exhaustivité des évolutions apportées au jeu de données, mais il a vocation à tracer les évolutions majeures de deux types :

- Sur les aspects techniques (ex : nouvelles colonnes, nouvelles valeurs possibles)
- Sur les aspects métiers (ex : nouvelle catégorie d’objet ou déchet)

## 2026-01-24

**Changements techniques :**

- Ajout de la colonne `rapporter` qui contient les sous-categories concernées par le geste rapporter de l'acteur : concerne essentiellement la consigne

## 2025-11-03

**Changements techniques :**

- Publication de la première version officielle des données :
  - Suppression de la mention BETA
- Publication de la documentation du jeu de données
- Publication de la politique de version du jeu de données
